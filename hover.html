Nesse capitulo aprendemos que, podemos, realizar interação com as "pseudo-class com as Div:hover", trazendo interção nos elementos, como ocutação de textos ou Drop box um menu de caracteres.

<style>
div > p{ display: none; }
div:hover > p{ display: block;
color: white;
background-color: red;
width: 300px;
} </style>
